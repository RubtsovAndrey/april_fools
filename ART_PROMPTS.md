# Dodo Reigns — Art Asset Generation Prompts

> **Generator:** Nano Banana Pro
> **Game orientation:** Landscape (16:9)
> **Visual style:** Reigns clone — geometric low-poly, dark corporate noir palette, ~30% more detail than original Reigns
> **IMPORTANT:** All assets must have SOLID backgrounds — no transparency, no PNG alpha.

---

## GLOBAL STYLE KEYWORDS

Append to every prompt:

```
geometric low-poly style, Reigns video game aesthetic, flat shading, angular simplified shapes, warm muted corporate palette, digital illustration, clean vector forms, no outlines, no gradients, solid color fills, slightly more detailed than Reigns original
```

**Core color palette:**
- Dark brown / card bg: `#3D2B1F`
- Warm beige / text panel: `#D4C5A9`
- Deep navy / night sky: `#1B1F3B`
- Muted orange / Dodo accent: `#C47B35`
- Olive green: `#5A6B3C`
- Dusty red / warning: `#A63D2F`
- Slate gray / tech: `#4A5568`
