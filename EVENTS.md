# DODO REIGNS — События и карточки

> **Этот файл — единый источник всех игровых событий.**
> Редактируйте его, после чего карточки в `cards.js` будут обновлены.

---

## Формат записи

```
### ID: Название события
- **Персонаж:** ключ_персонажа
- **Текст:** «Текст на карточке»
- **← Влево:** Название | Д:±X Б:±X К:±X Т:±X
- **→ Вправо:** Название | Д:±X Б:±X К:±X Т:±X
- **?:** ±X (загадочный ресурс, опционально)
- **Результат (←/→/any):** «Текст карточки-результата» (опционально)
- **Рок:** left/right — запускает роковую цепочку (опционально)
- **Флаги:** requires: {...}, sets: {...} (опционально)
- **Приоритет:** 1/2/3
```

## Ресурсы

| Код | Название | Ключ |
|-----|----------|------|
| **Д** | Финансы | `money` |
| **Б** | Бренд/Клиенты | `customers` |
| **К** | Команда/Партнёры | `partners` |
| **Т** | Технологии | `it` |
| **?** | Загадочный ресурс | `mystery` |

# РЕЖИМ ПИЦЦЕРИИ

> **Механика:** каждые 5-6 обычных карточек появляется «пиццерийная» карточка.
> Фон меняется на постоянный экран пиццерии (один и тот же всегда).
